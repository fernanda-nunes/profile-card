# Profile Card

<p><b>Esta é uma solução para o desafio do módulo de HTML e CSS, proposto pelo do curso DevQuest.</b></p>

## Visão Geral 

###  Projeto 
<b>O objetivo é criar uma página de cartão de perfil responsiva com uma única sessão, utilizando HTML e CSS.

###  Desafio
<b>O desafio consiste em construir uma página de cartão de perfil a partir dos designs fornecidos. A página deve ser responsiva, adaptando-se a diferentes tamanhos de tela.
